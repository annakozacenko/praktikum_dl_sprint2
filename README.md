Проект по автодополнению текстов на основе постов из датасета Sentiment140.

## Структура проекта

- `data/` — датасеты (raw, processed, train/val/test)
- `src/` — весь код (модель, обучение, оценка, утилиты)
- `models/` — веса обученной модели
- `solution.ipynb` — ноутбук с результатами и выводами

## Запуск

1. Установить зависимости: `pip install -r requirements.txt`
2. Обучить модель: `python src/lstm_train.py`
3. Открыть `solution.ipynb` для просмотра результатов
